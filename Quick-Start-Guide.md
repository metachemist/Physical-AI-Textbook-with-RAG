# Quick Start Guide

## What is this project?

This project creates an AI-native textbook for teaching Physical AI & Humanoid Robotics. The project is part of Hackathon I for Panaversity, focusing on embodied intelligence and the convergence of AI systems with the physical world. The textbook integrates an AI-powered RAG chatbot for personalized learning, adaptive content based on student background, and multi-language support for broader accessibility.

The project demonstrates AI-assisted content creation workflow and showcases modern development tools. It implements cutting-edge RAG architecture and creates reusable patterns for future textbooks, democratizing access to advanced robotics education and enabling students in developing countries to learn world-class content.

## What do I need to deliver?

### Core Deliverables (Required)
- **AI-Native Textbook**: Platform built with Docusaurus, deployed on GitHub Pages, focusing on Physical AI & Humanoid Robotics curriculum
- **Integrated RAG Chatbot**: Using OpenAI Agents SDK APIs, FastAPI backend, Neon Serverless Postgres, and Qdrant Cloud for answering questions about book content

### Bonus Features (Optional but Recommended)
- **Authentication System** (Better-Auth) with user signup/signin and background profiling
- **Content Personalization** with adaptive content based on user background
- **Localization** with Urdu translation capability
- **Reusable Intelligence** with Qwen Code Subagents and Agent Skills development

## How do I get started?

- Module 3: The AI-Robot Brain (NVIDIA Isaac™)
- Module 4: Vision-Language-Action (VLA)
